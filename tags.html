HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but dhey do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table raw include a row of data
th table header make a table header
td table data include a table data cell
header header make header block
div divider include a dvided block element
ol ordered list include an ordered list
ul unordered list include an unordered list
li list include a list item (ordered or unordered)
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <storng>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code include monospace text <code>code</code> code
span span include a certain lenghth of strings <span>String</span> String

Document Tags

Tag Name Purpose
html html make HTML skeleton
head head include head information
body body incude main body of the page
title title set page title
meta meta make settings of pages
